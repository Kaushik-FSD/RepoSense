Feature Modules

📁 Repo Assistant
RAG Embeddings Code Chunking
Ingest a repo, chunk code files intelligently, answer architecture/implementation questions via retrieval.
🔀 PR Summarizer
Diff Parsing Structured Output
Takes PR diff, retrieves related context, produces summary + affected modules + risk analysis.
🪵 Log Analyzer
Pattern Detection RAG
Ingests application logs, finds error patterns, suggests probable root causes via LLM.
📝 Release Notes
Structured Gen Prompt Engineering
Accepts commit messages / PR titles, outputs a structured, human-readable changelog.

API Endpoints (MVP)

MethodEndpointDescription
POST /repo/ingest Upload + parse + embed a repository (zip or folder path)
POST /repo/ask Ask a question about the ingested repo
POST /pr/analyze Submit PR diff → get summary + risk analysis
POST /logs/analyze Submit log text → get error patterns + root causes
POST /release/generate Submit commit list → get structured release notes
GET /health Health check
